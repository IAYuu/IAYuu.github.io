title: 序列化和反序列化
---
1. 打开并读 `json` 文件(字节流)
```go
fd, _ := os.Open("data.json")
bytes, _ := ioutil.ReadAll(fd)
```
2. 反序列化成类实例
```go
type Person struct {
        Name string    `json:"name"`
        Age  int       `json:"age"`
        Hobby []string `json:"hobbies"`
}
dx := Person{}
err = json.Unmarsel(bytes, &dx)
```
3. 序列化成字节流
```go
bytes, _ = json.Marshal(dx)
newfile, _ := os.Create("newfile.json")
_, err = newfile.Write(bytes)
```

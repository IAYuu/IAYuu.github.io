- 性能：差不多

## `Flask-SQLAlchemy`
- 提供高层 `ORM`
- 提供使用数据库原生 `SQL` 的底层操作

## 数据库操作
SQLALchemy提供支持
- 创建db对象
```
app.config['SQLALCHEMY_DATABASE_URI'] = \
           'sqlite:///' + os.path.join(basedir, 'data.sqlite')
app.config['SQLALCHEMY_TRACK_MODIFICATION'] = False
db = SQLAlchemy(app)
```
- 建表类
- 表类中添加关系
- 创建数据库表
使用 `Flask_SQLAlchemy` 根据模型创建数据库
```python
db.create_all()
# 删表
# db.drop_all()
```
